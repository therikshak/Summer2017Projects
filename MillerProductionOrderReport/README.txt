Instructions for Runnning the Report

1. Download the Miller Report, and save it with the name "pvso" to the folder

2. In the folder, double click on "MillerCoorsReportScript.vbs" and wait a few seconds.
	-nothing will pop-up, but the script just takes a few seconds to run

3. Open the excel file "MillerCoors_report.xlsm"

4. Make sure you are on the Pivot Table Sheet (if not, there should be tab at the bottom left of the excel window)

5. Click on the button that says "Update Data" and wait for the pivot table to update.
